Models

                
UNet
Standard U-Net architecture

                
Attention UNet
U-Net with attention gates

                
SegResNet
Segmentation ResNet

                
SwinUNETR
Swin Transformer-based UNETR

                
BasicUNet
Simplified U-Net architecture

                
BasicUNet++
UNet++ with nested skip connections for better feature reuse

                
DynUNet
Dynamic U-Net (nnU-Net style) with adaptive kernel sizes

                
VNet
Volumetric segmentation network with residual connections

                
HighResNet
High-resolution network designed for brain parcellation

                
UNETR
UNet with Vision Transformers as encoder

                
SegResNetVAE
SegResNet with variational autoencoder branch

                
SegResNetDS / DS2
SegResNet with deep supervision for better gradient flow

                
FlexibleUNet
U-Net with configurable backbone (EfficientNet-B0)

                
DiNTS
Differentiable Neural Architecture Search for segmentation

                
MedNeXt (S/M/B/L)
ConvNeXt-style medical image segmentation in four sizes

            


            

                
Datasets

                
Dataset
Basic in-memory dataset

                
CacheDataset
Caches entire dataset in memory for fast access

                
PersistentDataset
Persistent cache on disk with hash-based validity checking

                
SmartCacheDataset
Intelligent caching with automatic replacement

                
LMDBDataset
LMDB-backed persistent cache for fast random access I/O

                
CacheNTransDataset
Cache first N transforms for partial caching efficiency

                
ArrayDataset
Simple dataset from separate image/label arrays

                
ZipDataset
Combine multiple datasets together

                
GridPatchDataset
Grid-based patch extraction dataset

                
PatchDataset
Random patch extraction dataset

                
DecathlonDataset
Medical Segmentation Decathlon benchmark dataset

            


            

                
Training Params

                
Epochs
Number of training epochs
Default: 1

                
Batch Size
Number of samples per batch
Default: 4

                
Learning Rate
Optimizer learning rate
Default: 0.0001

                
Image Size
Image size for resizing (applies to all dimensions)
Default: 128

                
Workers
Number of data loading workers
Default: 0

                

                    
Optimizer

                    
Adam: Adaptive learning rate with momentum. Default.

                    
AdamW: Adam with decoupled weight decay.

                    
SGD: Stochastic Gradient Descent with momentum.

                    
Novograd: MONAI layer-wise adaptive optimizer. Good for large batch training.

                    
RMSprop: Root mean square propagation.

                

                

                    
LR Scheduler

                    
none: Constant learning rate. Default.

                    
cosine: Cosine annealing.

                    
step: Step decay at fixed intervals.

                    
plateau: ReduceLROnPlateau.

                    
warmup_cosine: Linear warmup + cosine decay.

                    
cosine_warm_restarts: Cosine with periodic warm restarts.

                    
polynomial: Polynomial decay schedule.

                

                

                    
Patience

                    
X: No early stopping (default).

                    
3-20: Enable early stopping after N epochs without improvement.

                

            


            

                
Preprocessing

                
MinMax Normalization
Scale intensity values to [0, 1] range.

                
Z-Score Normalization
Normalize to zero mean and unit variance.

                
Center Crop
Crop from center to target size.

                
Random Crop
Random crop to target size (training only).

            


            

                
Augmentation

                
Basic

                    
Rotate: Random rotation transform.

                    
Flip: Random flip (horizontal and/or vertical).

                

                
Spatial Transforms

                    
RandAffine: Combined rotation, scaling, and translation.

                    
Rand2DElastic / Rand3DElastic: Elastic deformation.

                    
RandCropByPosNegLabel: Balanced foreground/background sampling.

                    
CropForeground: Auto-crop to foreground.

                    
RandRotate90: Random 90-degree rotation.

                    
RandSpatialCropSamples: Multiple random crop samples.

                    
SpatialPad / BorderPad / DivisiblePad: Padding strategies.

                    
GridPatch: Grid-based patch extraction.

                

                
Intensity Transforms

                    
RandGibbsNoise / RandKSpaceSpikeNoise / RandBiasField: MRI artifacts.

                    
RandCoarseDropout / RandCoarseShuffle: Region-based augmentation.

                    
RandHistogramShift: Non-linear intensity mapping.

                    
RandShiftIntensity / RandScaleIntensity: Intensity offset/scaling.

                    
RandGaussianSmooth / RandGaussianSharpen: Blurring/sharpening.

                    
MaskIntensity / ClipIntensityPercentiles: Masking and clipping.

                    
ScaleIntensityRange / ThresholdIntensity: Range mapping.

                

            


             

                 
Metrics

                 
Dice
Dice coefficient for segmentation overlap.

                 
IoU
Intersection over Union (Jaccard Index).

                 
Hausdorff Distance
Maximum boundary distance.

                 
Surface Distance
Average surface distance.

                 
Surface Dice (NSD)
Normalized Surface Dice at tolerance threshold.

                 
Generalized Dice
Volume-weighted Dice for class imbalance.

                 
Confusion Matrix (F1)
F1, sensitivity, specificity, precision.

                 
F-Beta Score
Weighted harmonic mean of precision and recall.

                 
Panoptic Quality
PQ, SQ, RQ metrics.

             


            

                
Loss Functions

                
Dice Loss
Dice coefficient-based loss. Default.

                
Cross Entropy
Standard cross-entropy for multi-class.

                
Focal Loss
Focuses on hard examples for imbalanced data.

                
Dice + CE / Dice + Focal
Combined losses.

                
Generalized Dice
Weighted Dice for extreme class imbalance.

                
Generalized Wasserstein Dice
Label similarity-aware loss.

                
Tversky
Tunable FP/FN trade-off via alpha/beta.

                
Hausdorff DT / Log Hausdorff DT
Boundary-quality losses.

                
Soft clDice
Centerline Dice for tubular structures.

                
Masked Dice
Spatially masked Dice loss.

                
NACL
Neighbor-Aware Constrained Loss.

                
Asymmetric Unified Focal
Asymmetric focal formulation.

                
SSIM
Structural similarity loss.

                
Deep Supervision
Multi-scale weighted loss.

            


            

                
Inferer

                
Simple
Direct model forward pass. Default.

                
Sliding Window
Overlapping patches with Gaussian blending for large volumes.

                
Patch
Patch-based with splitter/merger pattern.

                
Saliency (GradCAM)
Class activation maps for interpretability.

                
Slice
Apply 2D model slice-by-slice on 3D volumes.

            


            

                
Device

                
CUDA
GPU acceleration

                
CPU
CPU computation

                

                    
Mixed Precision

                    
no: Full precision (32-bit). Default.

                    
fp16: Half precision. Faster, less stable.

                    
bf16: Brain float. Better stability than fp16.
